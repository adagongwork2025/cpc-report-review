## 🚀 5 分鐘快速啟動

### 步驟 1: 建立資料庫

```bash
# 建立資料庫
createdb cpc_reports

# 建立使用者
psql -c "CREATE USER cpc_user WITH PASSWORD 'your_password';"
psql -c "GRANT ALL PRIVILEGES ON DATABASE cpc_reports TO cpc_user;"

# 執行 Schema
cd cpc-reports-backend
psql cpc_reports < schema.sql
```

### 步驟 2: 安裝 Python 套件

```bash
# 建立虛擬環境
python3 -m venv venv
source venv/bin/activate

# 安裝套件
pip install -r requirements.txt
```

### 步驟 3: 設定環境變數

```bash
# 複製範例檔案
cp .env.example .env

# 編輯 .env（修改密碼）
# DATABASE_URL=postgresql://cpc_user:your_password@localhost:5432/cpc_reports
```

### 步驟 4: 匯入資料

```bash
# 從 HTML 檔案匯入資料到資料庫
python migrations/migrate_html_data.py
```

### 步驟 5: 啟動 API

```bash
# 開發模式（自動重載）
uvicorn app.main:app --reload

# 開啟瀏覽器測試
# http://localhost:8000/docs
```

## 📊 資料庫結構

```
detection_logs (偵測記錄)
  ├── 對應 HTML 的 DATA 陣列
  └── 包含：時間、訊息、人員ID、圖片、影片

review_actions (審核歷史)
  ├── 記錄每次審核操作
  └── 支援審核歷史追蹤

review_states (當前狀態)
  ├── 快取表，快速查詢最新狀態
  └── pending | confirmed | deleted

export_logs (匯出記錄)
  └── 記錄報表匯出歷史
```

## 🔌 主要 API 端點

### 偵測記錄
```bash
# 取得 4/7 高處作業的偵測記錄
curl http://localhost:8000/api/v1/detection-logs/2026-04-07/高處作業
```

### 審核
```bash
# 批次提交審核
curl -X POST http://localhost:8000/api/v1/reviews/actions/bulk \
  -H "Content-Type: application/json" \
  -d '{
    "date": "2026-04-07",
    "category": "高處作業",
    "actions": [
      {"detection_log_id": 1, "action": "confirmed"}
    ],
    "reviewer_ip": "192.168.1.1"
  }'
```

### 報告
```bash
# 取得已審核的報告
curl http://localhost:8000/api/v1/reports/2026-04-07/高處作業
```

### 統計
```bash
# 取得有資料的日期列表
curl http://localhost:8000/api/v1/stats/dates?year=2026&month=4
```

## ✨ 核心功能

### 1. 偵測記錄管理
- 從 API 取得偵測資料
- 取代 HTML 內嵌的 DATA 陣列
- 支援日期+類別篩選

### 2. 審核流程
- 批次提交審核（confirmed/deleted）
- 審核歷史追蹤（誰、何時、做了什麼）
- 審核狀態快取（加速查詢）

### 3. 報告產生
- 只顯示已確認的違規項目
- 支援備註功能
- 記錄最後審核時間

### 4. 統計功能
- 日期列表（替代 index.html 的 REPORT_DATA）
- 審核進度統計
- 跨日期統計分析

```sql
-- 連線到資料庫

SELECT date, category, COUNT(*)
FROM detection_logs
```

### 2. 測試 API

```bash

curl http://localhost:8000/health

# API 文件
open http://localhost:8000/docs
```

### 3. 比對資料完整性

```bash
# 檢查 HTML 和資料庫的資料筆數是否一致
python -c "
from pathlib import Path
import re, json

html_count = 0
for html_file in Path('2026').rglob('*_審核.html'):
    content = html_file.read_text()
    match = re.search(r'const DATA = (\[.*?\]);', content, re.DOTALL)
    if match:
        data = json.loads(match.group(1))
        html_count += len(data)

print(f'HTML 總筆數: {html_count}')
"

# 然後查詢資料庫
# psql cpc_reports -c "SELECT COUNT(*) FROM detection_logs;"
# 應該要相同
```
