========================================
中油偵測報告系統 - 前端更新部署說明
========================================

【更新內容】
將審核資料從「瀏覽器 localStorage」改為「伺服器資料庫」儲存，
讓不同電腦/瀏覽器看到相同的審核結果。

【部署步驟】

1. 備份現有檔案
   將 https://cpcreportlist.intemotech.com/ 的現有檔案備份

2. 解壓縮部署包
   將 cpc_v2_部署包_XXXXXXXX.zip 解壓縮

3. 上傳檔案
   將解壓縮後 cpc-reports-backend-v2/ 資料夾內的所有檔案
   上傳到網站根目錄，覆蓋現有檔案：

   - index.html （覆蓋）
   - 2026/ 資料夾（覆蓋）

4. 確認後端 API 運作中
   API 位址：http://192.168.53.96:8001/api/v1
   測試：curl http://192.168.53.96:8001/api/v1/stats/dates?year=2026

【注意事項】
- 後端 API 必須持續運作
- 前端會從 API 讀取/寫入審核資料
- 舊的 localStorage 資料不會自動遷移（已另外處理）

【測試方式】
部署完成後，開啟任一審核頁面：
1. 確認或刪除一筆資料
2. 重新整理頁面，確認狀態有保留
3. 用另一台電腦開啟同一頁面，確認資料一致
