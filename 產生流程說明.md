## 完整產生流程

### Step 1: 取得 API 資料

```python
#!/usr/bin/env python3
import requests
import json
import re
import ssl
import urllib.request

API_URL = "https://apigatewayiseek.intemotech.com/vision_logic/logs/notify/filter"

def fetch_data(camera_id, date_str, start_time="08:00:00", end_time="13:30:00"):
    """取得指定 camera 和日期的資料"""
    all_logs = []
    page = 1
    per_page = 1000

    ctx = ssl.create_default_context()
    ctx.check_hostname = False
    ctx.verify_mode = ssl.CERT_NONE

    while True:
        payload = {
            "start_time": f"{date_str}T{start_time}",
            "end_time": f"{date_str}T{end_time}",
            "camera_id": camera_id,
            "per_page": per_page,
            "page": page
        }

        data = json.dumps(payload).encode('utf-8')
        req = urllib.request.Request(API_URL, data=data, headers={
            'Content-Type': 'application/json',
            'User-Agent': 'Mozilla/5.0'
        })

        with urllib.request.urlopen(req, timeout=60, context=ctx) as resp:
            result = json.loads(resp.read().decode('utf-8'))
            logs = result.get('logs', [])
            all_logs.extend(logs)

            if page >= result.get('total_pages', 1) or not logs:
                break
            page += 1

    return all_logs
```

### Step 2: 去重處理

```python
def deduplicate_data(items):
    """根據日期、人員ID、違規類型去重"""
    seen = set()
    unique = []

    for item in items:
        time_str = item.get('time', '')
        date_key = time_str[:10] if 'T' in time_str else ''

        mesg = item.get('mesg', '')
        ids = sorted(re.findall(r'ID:(\d+)', mesg))
        ids_key = ','.join(ids)

        log_type = item.get('type', '')

        # 建立唯一鍵：日期_人員IDs_類型
        key = f"{date_key}_{ids_key}_{log_type}"

        if key not in seen:
            seen.add(key)
            unique.append(item)

    return unique
```

def filter_harness(items):

    return [item for item in items if item.get('type') != 'harness']

### Step 4: 處理資料格式

```python
def process_data(raw_data):
    """處理 API 原始資料為頁面所需格式"""
    processed = []
    for item in raw_data:
        # 提取 video path
        video_key = item.get('video_key', '')
        if 'path:' in video_key:
            path_match = re.search(r'path:([^\s,]+)', video_key)
            if path_match:
                video_key = path_match.group(1)

        # 提取時間
        time_str = item.get('time', '')
        time_display = time_str.split('T')[1][:8] if 'T' in time_str else ''

        # 提取 person IDs
        mesg = item.get('mesg', '')
        ids_match = re.findall(r'ID:(\d+)', mesg)
        person_ids = ', '.join(ids_match) if ids_match else ''

        processed.append({
            'id': item.get('id'),
            'time_display': time_display,
            'mesg': mesg,
            'type': item.get('type', ''),
            'person_ids': person_ids,
            'image_url': f"https://apigatewayiseek.intemotech.com/vision_logic/logs/notify/image?key={item.get('key', '')}",
            'video_url': f"https://apigatewayiseek.intemotech.com/vision_logic/video?key={video_key}"
        })
    return processed
```

## 常見錯誤與解決方法

### 問題 1: 統計數據顯示負數

**原因**：頁面中有預設的 `deletedIds`

**解決**：
```python
html = re.sub(r'let deletedIds = new Set\(\[.*?\]\);', 'let deletedIds = new Set();', html, flags=re.DOTALL)
```

### 問題 2: 報告頁面沒有資料

**原因**：REPORT_KEY 不正確，導致讀取錯誤的 localStorage

**檢查**：
```javascript
// 審核頁面
const REPORT_KEY = 'reportData_' + DATE + '_' + CATEGORY;

// 報告頁面
const REPORT_KEY = 'reportData_2026-04-13_高處A';  // 必須一致
```

### 問題 3: 點擊「偵測報告」出現 404

**原因**：REPORT_PATH 路徑錯誤

**解決**：
```javascript
const REPORT_PATH = '高處A_報告_管理版.html';  // 確保檔案存在
```

### 問題 4: 兩個 Camera 的頁面顯示相同圖片

**原因**：使用了模板替換方法，資料沒有正確分離

**解決**：使用完整 HTML 產生方法，從 4/07 模板開始，完整替換所有內容
