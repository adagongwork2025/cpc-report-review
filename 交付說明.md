## 給工程師的重點提示

### ✅ 必須事項

1. **啟用 HTTPS**
   - 系統需要呼叫外部 API，瀏覽器會要求 HTTPS
   - 沒有 HTTPS 影片和圖片都無法載入

2. **檢查網路連線**
   - 確保伺服器可以存取：`apigatewayiseek.intemotech.com`
   - 測試方式：`curl https://apigatewayiseek.intemotech.com/vision_logic/video?key=/test`

3. **靜態檔案伺服器即可**
   - 不需要 Node.js、PHP、Python 等後端
   - Nginx、Apache、S3、GCS 都可以
   - 只要能提供靜態檔案服務即可

### 📋 部署方式建議

**方案 A：Nginx（推薦）**
```nginx
server {
    listen 443 ssl;
    server_name cpc-detection.example.com;

    root /var/www/cpc-detection;
    index index.html;

    ssl_certificate /path/to/cert.pem;
    ssl_certificate_key /path/to/key.pem;
}
```

**方案 B：AWS S3 + CloudFront**
1. 上傳到 S3 Bucket
2. 啟用靜態網站託管
3. 用 CloudFront 提供 HTTPS

**方案 C：Azure Blob Storage**
1. 建立 Storage Account
2. 啟用靜態網站
3. 設定 CDN

### 🧪 測試步驟

1. **基本測試**
   ```bash
   # 確認主頁可存取
   curl https://your-domain.com/

   # 確認資料可存取
   curl https://your-domain.com/2026/04/07/高處作業_審核.html
   ```

2. **功能測試**
   - 開啟主頁面，應該看到日期列表
   - 點擊「審核」連結，進入審核頁面
   - 點擊影片播放按鈕，影片應正常播放
   - 完成審核後，報告按鈕應從灰色變成綠色

3. **瀏覽器測試**
   - Chrome 90+
   - Safari 14+
   - Firefox 88+
   - Edge 90+

## 常見問題 Q&A

### Q1：需要安裝什麼套件嗎？
**A**：不需要！這是純靜態網站，只需要 Web Server 即可。

### Q2：資料存在哪裡？
**A**：所有資料都嵌入在 HTML 檔案的 JavaScript `DATA` 陣列中，使用 localStorage 儲存審核結果。

### Q3：如何新增新的日期資料？
**A**：
- 手動方式：複製現有 HTML，修改 DATA 陣列
- 自動方式：執行 `scripts/generate_daily_review.py`

### Q4：可以改樣式嗎？
**A**：可以！所有 CSS 都在 HTML 的 `<style>` 標籤內，直接修改即可。

### Q5：如何備份審核記錄？
**A**：審核記錄存在瀏覽器的 localStorage 中，建議定期匯出或改用後端儲存。

### Q6：影片為什麼無法播放？
**A**：
1. 檢查是否使用 HTTPS
2. 檢查網路是否能存取 apigatewayiseek.intemotech.com
3. 查看瀏覽器 Console 的錯誤訊息

## 後續維護

### 新增每日資料

**自動化方式（推薦）**：
```bash
cd scripts/
python3 generate_daily_review.py
```

腳本會自動：
- 從 API 取得當日資料
- 產生審核和報告頁面
- 更新 index.html

**手動方式**：
1. 複製類似日期的檔案
2. 修改 DATA 陣列
3. 更新 index.html 的 REPORT_DATA

### 清除審核記錄

在瀏覽器 Console 執行：
```javascript
// 清除所有審核記錄
for (let key in localStorage) {
  if (key.startsWith('reportData_')) {
    localStorage.removeItem(key);
  }
}
```
